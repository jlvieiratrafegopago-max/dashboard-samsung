# 📊 Samsung Global Sales Dashboard

Este é um projeto de **Business Intelligence (BI)** desenvolvido para analisar o desempenho global de vendas da Samsung. A aplicação transforma dados brutos em insights estratégicos acionáveis, permitindo uma visão clara de faturamento, satisfação do cliente e tendências de mercado.

🚀 **[CLIQUE AQUI PARA ACESSAR O DASHBOARD AO VIVO](https://dashboard-samsung-6cr8op8qdjm2hsregvwthy.streamlit.app/)**

---

## 🎯 Objetivo do Projeto
Demonstrar como a combinação de **Python** e **Streamlit** pode substituir relatórios estáticos por ferramentas dinâmicas que auxiliam na tomada de decisão em tempo real para grandes empresas e agências de marketing.

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem core para processamento de dados.
* **Streamlit**: Framework para criação da interface web interativa.
* **Pandas**: Biblioteca para manipulação e limpeza de grandes conjuntos de dados (CSV).
* **Plotly**: Criação de gráficos dinâmicos e mapas coropléticos.

## 📈 Funcionalidades Principais
* **KPIs Dinâmicos**: Visualização instantânea de Receita Total, Unidades Vendidas, Ticket Médio e Avaliação dos Clientes.
* **Visão Geográfica**: Mapa interativo mostrando a distribuição de faturamento por país.
* **Análise de Tendências**: Gráfico de linha temporal para identificar sazonalidade e picos de vendas.
* **Mix de Produtos**: Gráfico de pizza detalhando a participação de cada categoria (Smartphones, Tablets, etc.) no faturamento.
* **Filtros Inteligentes**: Filtros por Região e Categoria que atualizam todo o dashboard automaticamente.
